## 🙋 마이페이지 (MyTab) — 전면 개편
- **2탭 구조**(크롬 탭식): `경기 참가` / `운동·연습 기록` 으로 분리. 프로필·요약 블럭은 공통 상단 고정.
- **프로필 사진 업로드**: 아바타(📷) 탭 → 이미지 선택·미리보기.
- **참가 예정 경기 카드 개편**: 좌측 **D-day 크게** + 대회명·🏷참가종목·🎒준비물 + 우측 **🔔 알림 종(on/off)**.
  - 종 탭 → **경기 알림 설정 팝업**: 출전 5경기(heat) 전 · 출전 코트 배정 · 경기 시작 · 기록영상 업로드 토글
  - **소리/진동** 선택 + **알람음**(기본음·호루라기·종소리·박수·응원 함성) + "이런 알림이 울려요" 미리보기
  - 참가 신청이 없으면 *"현재 경기 참가 신청이 없습니다"* + **경기 참가 직접 등록**(신규 페이지)
- **참가한 대회 리스트**: 펼치면 **종목별 결과 + 🏅수상 + 🎬영상 + ▶️영상 보기 + 🔗공유 + 📋링크 복사**.
- 운동·연습 탭: 기록 그래프(일/주/월)·연습 vs 실전 분석·AI 분석·주간 리포트·훈련 플랜·연습일지.

## 🎬 대회 영상 (VodTab) — 푸터명 `기록영상` → `대회 영상`
- **대회·일차별 그룹화**(예: 6/7 1일차, 6/8 2일차) + 그룹 헤더에 **대회명** 표시.
- **⭐ 내 선수·팔로우 영상 상단 우선** 노출.
- **검색**(이름·코트·HIT) + **필터 아코디언**(대회·일차 / 종목·수상) → **조회하기** 버튼으로 적용.
- **로딩 중 = 위플레이 로고**(펄스 애니메이션).
- 영상 선택 시 **어느 대회 기록인지** 재생 모달에 표시.

## 📺 중계 (LiveTab / CourtSheet / CourtCard)

- 홈/중계 **LIVE 배지 = 실제 송출 중일 때만** 표시(`liveNow`).
